Learn JavaScript

A beginner-friendly interactive playground to learn and practice JavaScript basics.

JavaScript Learning Path

  1. Introduction & Setup
    What is JavaScript? How to run JS in the browser.
  2. Variables & Data Types
    let, const, var, strings, numbers, booleans.
  3. Operators & Expressions
    +, -, *, /, %, =, ==, ===, and more.
  4. Control Flow
    if, else, else if, switch.
  5. Loops
    for, while, do...while.
  6. Functions
    Declaring and calling functions, parameters, return values.
  7. Arrays & Objects
    Basic array and object usage.
  8. DOM Basics (optional)
    How to change web pages with JavaScript.
  9. Practice Projects
    Try making a calculator, to-do list, or your own ideas!
Work through each step, try the examples, and experiment in the code editor below!

Step-by-Step Examples & Exercises

1. Introduction & Setup
Example: 
// This is a comment\nconsole.log('JavaScript is running!');
Exercise:
  • Change the message to your own greeting and run the code.
2. Variables & Data Types
Example: 
let age = 15;\nconst name = \"Alex\";\nconsole.log(name + ' is ' + age + ' years old.');
Exercise:
  • Create a variable for your favorite color and print it.
3. Operators & Expressions
Example: 
let sum = 7 + 3;\nlet isEqual = (sum === 10);\nconsole.log('Sum:', sum);\nconsole.log('Is sum 10?', isEqual);
Exercise:
  • Try using -, *, /, and % operators with numbers and print the results.
4. Control Flow
Example: 
let score = 85;\nif (score >= 80) {\n  console.log('Great job!');\n} else {\n  console.log('Keep practicing!');\n}
Exercise:
  • Change the score value and see what message you get.
5. Loops
Example: 
for (let i = 1; i <= 5; i++) {\n  console.log('Number:', i);\n}
Exercise:
  • Write a loop that prints even numbers from 2 to 10.
6. Functions
Example: 
function greet(name) {\n  console.log('Hello, ' + name + '!');\n}\ngreet('Sam');
Exercise:
  • Create a function that multiplies two numbers and prints the result.
7. Arrays & Objects
Example: 
let fruits = ['apple', 'banana', 'cherry'];\nconsole.log(fruits[1]); // banana\nlet person = { name: 'Lina', age: 20 };\nconsole.log(person.name);
Exercise:
  • Create an array of three animals and print the second one.
8. DOM Basics (optional)
Example: 
// This only works in a real web page, not the code runner above!\ndocument.body.style.background = '#ffe4e1';
Exercise:
  • Try changing the page background color using JavaScript (open browser console to test).
9. Practice Projects
Ideas:
  • Make a simple calculator (add, subtract, multiply, divide two numbers).
  • Build a to-do list (array of tasks, add/remove items).
  • Write a program that asks for a name and prints a greeting.
